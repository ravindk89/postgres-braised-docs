This part describes the client programming interfaces distributed with PostgreSQL.
Each of these chapters can be read independently.
There are many external programming interfaces for client programs that are distributed separately.
They contain their own documentation ([H. External Projects](braised:ref/external-projects) lists some of the more popular ones).
Readers of this part should be familiar with using SQL to manipulate and query the database (see [II. The SQL Language](braised:ref/sql)) and of course with the programming language of their choice.
