In the sections that follow, we will discuss how you can extend the PostgreSQL SQL query language by adding:

-   functions (starting in [Section 36.3](braised:ref/xfunc))
-   aggregates (starting in [Section 36.12](braised:ref/xaggr))
-   data types (starting in [Section 36.13](braised:ref/xtypes))
-   operators (starting in [Section 36.14](braised:ref/xoper))
-   operator classes for indexes (starting in [Section 36.16](braised:ref/xindex))
-   packages of related objects (starting in [Section 36.17](braised:ref/extend-extensions))
